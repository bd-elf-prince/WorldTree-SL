계정 허브

인증과 권한 범위가 확정되면 플레이어 프로필, 장비, 강화 기록을 이 도트 카드 UI에 연결합니다.

프로필

SL 아바타 UUID와 공개 닉네임 기준으로 정보를 표시합니다.

장비

검 단계, 속성, 강화 상태를 픽셀 카드로 확인합니다.

기록

강화 요청, 성공·실패, 보상 지급 내역을 서버 ledger와 맞춥니다.

플레이 가이드

신규 유저가 바로 이해할 수 있는 핵심 루프입니다.

01 · TOUCH

HUD와 NPC를 터치해 카드 선택, 강화 요청, 보상 확인을 진행합니다.

02 · ENCHANT

선택 카드와 장비 단계에 따라 서버가 강화 결과를 판정합니다.

03 · GROW

성공 기록을 쌓고 더 높은 검 단계와 랭킹에 도전합니다.

공지

운영 공지와 업데이트 노트를 표시하는 도트 게시판입니다.

포털 픽셀 디자인 적용

공식 홈페이지 전체를 픽셀 RPG 스타일로 재구성했습니다.

서버 API 연동 대기

랭킹과 계정 데이터는 서버팀 API 계약 확정 후 실제 데이터로 교체합니다.

리소스

공개 가능한 아트, UI, 운영 문서를 모으는 허브입니다. 비공개 서버·시트 자료와 분리합니다.

공개 리소스 정책

웹팀은 공개 홈페이지 코드와 공개 표시 자료만 담당합니다.

배포 상태

GitHub Pages main/root 배포 기준으로 운영합니다.
