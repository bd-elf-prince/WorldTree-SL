계정 허브

인증과 권한 범위가 확정되면 플레이어 프로필, 보유 장비, 최근 강화 기록을 이곳에 안전하게 표시합니다.

프로필

SL 아바타 UUID와 공개 닉네임을 기준으로 플레이어 정보를 표시합니다.

장비

검 단계, 속성, 강화 상태를 카드형 UI로 확인할 수 있게 준비했습니다.

기록

강화 요청, 성공·실패, 보상 지급 내역을 서버 ledger와 맞춰 보여줄 예정입니다.

플레이 가이드

신규 유저가 바로 이해할 수 있도록 핵심 루프를 짧게 정리했습니다.

01 · Touch

HUD와 NPC를 터치해 카드 선택, 강화 요청, 보상 확인을 진행합니다.

02 · Enchant

선택한 카드와 현재 장비 단계에 따라 강화 결과가 서버에서 판정됩니다.

03 · Grow

성공 기록이 누적되면 더 높은 검 단계와 랭킹 경쟁에 도전합니다.

공지

운영 공지와 업데이트 노트를 표시하는 영역입니다.

포털 정적 버전 공개

GitHub Pages 배포가 활성화되면 이 페이지가 공식 홈페이지로 표시됩니다.

서버 API 연동 대기

랭킹과 계정 데이터는 서버팀 API 계약 확정 후 실제 데이터로 교체합니다.

리소스

공개 가능한 아트, UI, 운영 문서를 모으는 허브입니다. 비공개 서버·시트 자료와 분리해 관리합니다.

공개 리소스 정책

웹팀은 공개 홈페이지 코드와 공개 표시 자료만 담당합니다. 서버 인증, DB, Google Sheet 연동, 비공개 로직은 서버팀 영역으로 유지합니다.
