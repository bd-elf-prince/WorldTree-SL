# WorldTree-SL

Official portal and first playable web-game MVP for WorldTree.

## Live files

- `index.html` — official portal / launcher
- `game.html` — playable web-game MVP

## Current MVP loop

1. Open the portal.
2. Start the game.
3. Use action points to explore the world map.
4. Chop forest nodes for Wood.
5. Visit caves, camps, chests, snowfield, and ruins.
6. Return / reincarnate to convert run progress into Rebirth Points.
7. Continue the next run with stronger base stats.

## Publish target

Use GitHub Pages with:

- Source: Deploy from branch
- Branch: `main`
- Folder: `/root`

After Pages is enabled, the portal should open from the repository Pages URL and `game.html` should be reachable from the portal start button.
